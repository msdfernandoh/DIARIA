❓ Dúvidas gerais

Perguntas
frequentes

O app é mesmo gratuito ou tem pegadinha? +
Para profissionais e contratantes: gratuito para sempre, sem limitação de uso. Destaques são opcionais e pagos. Para parceiros: gratuito por 3 meses com meta simples de engajamento; depois R$29/mês se não cumprir a meta.
Como o app garante a segurança dos pagamentos? +
O app não processa pagamentos — eles acontecem diretamente entre as partes (Pix, dinheiro, transferência). O chat do app registra tudo que foi combinado. O sistema de avaliações públicas cria responsabilidade natural para os dois lados.
Qual a diferença entre instância local e o app nacional? +
O banco de dados é único e nacional — todos compartilham as mesmas vagas e candidatos. A instância local adiciona uma camada de identidade regional: logo, nome, cor e filtro geográfico padrão para a cidade. O usuário pode a qualquer momento expandir para ver todo o Brasil.
É possível usar para emprego fixo CLT também? +
Sim. O app tem três abas: Trabalho (diárias), Emprego (CLT, PJ, estágio) e Oportunidades (franquias, parcerias, negócios). Cada perfil de usuário vê o que é mais relevante para ele.
Funciona para home office? +
Sim. Vagas presenciais aparecem com localização. Vagas de home office aparecem para todos — independente da cidade. O profissional escolhe se quer ver apenas locais, apenas remotas, ou as duas.
